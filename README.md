# aoc-rust

### download

```shell
(mkdir --parents data/year2023/day05 && cd data/year2023/day05 && aoc download --overwrite --year 2023 --day 05)
```

### solve all challenges

```shell
cargo run
```

### clippy

```shell
cargo clippy --all-targets --all-features
```

### benchmark

```shell
cargo bench --benches year2023::day05::part2
```
